Pathfind.r

Pathfindr is a free, lightweight web application that generates a user-friendly layout to simplify group trip planning. Users can create, edit, and share detailed itineraries for better coordination. Core features include exporting itineraries as ICS calendar files and PDFs. This project combines front-end and back-end design, cloud stack development, and database management.
